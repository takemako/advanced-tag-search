# Advanced Tag Search v1.7.0 - リリースノート

## 📋 概要

v1.7.0 は、検索体験と管理画面の使い勝手を大きく向上させるアップデートです。
キーワードによるサイト内検索、タグカテゴリーの動的な追加・削除、クイックリンクのカテゴリー選択、
そして絞り込み結果の記事数表示など、訪問者・管理者の双方にメリットのある機能を追加しました。

- **対象バージョン:** 1.7.0
- **必須環境:** WordPress 6.0 以上 / PHP 7.4 以上
- **推奨環境:** WordPress 6.8 / PHP 8.2

---

## ✨ 新機能

### 1. キーワード検索（サイト内検索）

検索モーダルの上部に「キーワードで検索」入力欄を追加しました。

- 入力したキーワードで WordPress 標準のサイト内検索（`?s=キーワード`）を実行
- Enter キーでも検索可能
- タグ・カテゴリーの選択と**組み合わせた絞り込み**にも対応

### 2. タグカテゴリーの動的UI（標準3種類＋追加ボタン）

タグカテゴリー設定を、ブロック単位で増減できる動的UIに変更しました。

- 標準は **3カテゴリー**（ジャンル／エリア／キーワード）
- **「＋ カテゴリーを追加」** ボタンで項目を追加
- 各ブロックの **「このカテゴリーを削除」** で削除
- タグは **WordPress 登録済みタグからチェックボックスで選択**（「全選択」「全解除」対応）

### 3. クイックリンクのカテゴリー選択

クイックリンクを、ブログのカテゴリーから表示するものを選ぶ方式に変更しました。

- 投稿のあるカテゴリーを **チェックボックスで選択**（「全選択」「全解除」対応）
- 未設定なら投稿のある全カテゴリーを表示、すべて未選択なら非表示
- リンク先は各カテゴリーのアーカイブページ

### 4. 絞り込みボタンに記事数を表示

絞り込みボタンの数字を、選択条件に一致する **記事数** に変更しました。

- 選択中のタグ・カテゴリー（AND）＋キーワードに一致する公開記事数を Ajax で取得
- 「選択した条件で絞り込む **(N件)**」と表示し、選択・入力のたびに更新

### 5. カテゴリー絞り込みの表示選択

検索モーダルの「カテゴリーから絞り込む」に表示するカテゴリーを、管理画面でチェックボックス選択できるようにしました（「全選択」「全解除」対応）。

---

## 🔧 改善

- タグ・カテゴリー・クイックリンクの各選択リストに「全選択」「全解除」ボタンを追加
- PHP / WordPress の動作要件チェックを追加（要件未達時は有効化を中止し管理画面に通知）。設定画面に「動作環境」セクションを追加
- 作者表記・推奨バージョンなどのドキュメントを整理

---

## 🔼 アップグレード時の注意

- 旧バージョンの「リンクテキストとURLを手入力するクイックリンク」は廃止され、カテゴリーからの選択方式になりました。
- タグはWordPressに登録済みのタグから選択します。表示したいタグは事前に「投稿 → タグ」で作成してください。

---

## 📦 インストール / アップデート

1. `advanced-tag-search.zip` をダウンロード
2. WordPress 管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ZIP を選択してアップロードし、「今すぐインストール」→「プラグインを有効化」

詳細は `INSTALL.md` / `USAGE_GUIDE.md` を参照してください。
