# Advanced Tag Search v1.8.1 - リリースノート

## 📋 概要

v1.8.1 は不具合修正リリースです。日本語スラッグのタグ・カテゴリー（例: 「レビュー」）で絞り込むと検索結果が0件になる問題を修正しました。

- **対象バージョン:** 1.8.1
- **必須環境:** WordPress 6.0 以上 / PHP 7.4 以上
- **推奨環境:** WordPress 6.8 / PHP 8.2

---

## 🐛 修正した不具合

### 日本語スラッグのタグ・カテゴリーで0件になる問題

「レビュー」のように名前が日本語のタグ・カテゴリーは、スラッグがURLエンコードされた文字列（例: `%e3%83%ac...`）になります。

- **症状:** 該当タグ・カテゴリーで絞り込むと、記事が存在しても結果が **0件** になる（英語スラッグのタグは正常）。絞り込みボタンの件数表示は件数が出るのに、実際の結果が0件になる不整合も発生していました。
- **原因:** 単一タグの絞り込みを WordPress 標準の `?tag=スラッグ` 処理に委ねていたため、`tag` クエリ変数が再サニタイズされ、日本語スラッグが壊れてタグアーカイブが空になっていました。複数タグ・カテゴリー側も `sanitize_title()` でスラッグを壊していました。
- **対応:** タグ・カテゴリーの絞り込み処理を、件数表示Ajaxと同じ `tax_query`（`field=slug`・`operator=AND`）に統一しました。これにより日本語スラッグでも確実に一致し、**件数表示と実際の検索結果が一致**します。

## 🔼 アップグレード時の注意

- 設定の変更は不要です。プラグインを更新するだけで修正が適用されます。

---

## 📦 インストール / アップデート

1. `advanced-tag-search.zip` をダウンロード
2. WordPress 管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ZIP を選択してアップロードし、「今すぐインストール」→「プラグインを有効化」
