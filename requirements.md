# WordPress検索プラグイン要件定義

## 概要
Kawagoe.funのような検索機能を持つWordPressプラグインを開発する。

## 機能要件

### 1. 検索窓の表示
- ブログのトップページに検索窓を配置
- プレースホルダー: 「タグから探してみる」
- 検索ボタン(虫眼鏡アイコン)を右側に配置

### 2. 検索窓下のクイックリンク
- カテゴリーやタグへのクイックリンクを表示
- 例: 「川越のお店一覧」「特集記事一覧」「川越氷川神社」「川越でスイーツ」「サイトマップ」

### 3. モーダルウィンドウ機能
検索窓をクリック/フォーカスすると、モーダルウィンドウが開く

#### モーダルの構成:
- **タイトル**: 「タグ検索」
- **閉じるボタン**: 右上にXボタン

#### タグのカテゴリー分類:
1. **ジャンルから絞り込む**
   - 食べ歩き、レストラン、カフェ、スイーツ、居酒屋、バー
   - 宿泊施設、温泉・銭湯、展示、体験、お土産
   - 神社・寺院、観光スポット、お祭り

2. **エリアから絞り込む**
   - 蔵造りの街並み周辺、時の鐘周辺、菓子屋横丁周辺、氷川神社周辺
   - 川越市役所周辺、昭和の街周辺、大正浪漫夢通り周辺、川越駅
   - 本川越駅・川越市駅、クレアモール周辺、喜多院周辺、川越城周辺
   - 南古谷駅方面、南大塚駅方面、新河岸駅方面、西川越駅方面
   - 霞ヶ関駅方面

3. **キーワードから絞り込む**
   - 芋、鰻、団子、蕎麦、太麺焼きそば、カレー、うどん、肉
   - ラーメン、コーヒー、ケーキ、アイス、かき氷、お茶、お酒

4. **時間帯から絞り込む**
   - モーニング、ランチ、ディナー

5. **季節から絞り込む**
   - 春、夏、秋、冬

6. **その他から絞り込む**
   - おすすめまとめ、観光コース、月がきれい、kokoradi

#### ボトムボタン:
- 「選択したタグで絞り込む」ボタン(ピンク色)

### 4. 検索機能の動作
- **カテゴリークリック**: 該当カテゴリーの記事一覧ページへ遷移
- **タグクリック**: 該当タグの記事一覧ページへ遷移
- **自由検索**: WordPress標準の検索機能と同じ動作
- **複数タグ選択**: 選択したタグでAND/OR検索が可能

## デザイン要件

### 検索窓
- 背景色: 白系
- 角丸のデザイン
- シャドウ効果あり
- レスポンシブ対応

### モーダルウィンドウ
- 背景: 半透明のオーバーレイ
- モーダル本体: 白背景、角丸
- タグ: ハッシュタグ形式(#付き)
- タグのホバー効果
- セクションごとに区切り線

### カラースキーム
- 検索ボタン: 赤茶色系(#A0616A程度)
- 絞り込みボタン: ピンク系
- タグ: グレー系背景、黒文字

## 技術要件

### プラグイン構成
- PHPによるプラグイン本体
- JavaScript(jQuery)によるモーダル制御
- CSSによるスタイリング
- ショートコード対応

### WordPress連携
- カテゴリー取得: `get_categories()`
- タグ取得: `get_tags()`
- 検索機能: WordPress標準の`WP_Query`を使用

### ショートコード
- `[advanced_search]` - 検索窓を表示
- `[search_quick_links]` - クイックリンクを表示

## インストール・使用方法
1. プラグインをWordPressの`wp-content/plugins/`にアップロード
2. 管理画面でプラグインを有効化
3. ショートコードをトップページに配置
4. 管理画面でタグのカテゴリー分類を設定(オプション)
