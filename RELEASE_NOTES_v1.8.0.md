# Advanced Tag Search v1.8.0 - リリースノート

## 📋 概要

v1.8.0 では、検索モーダル内のブロックの**表示順を管理画面から変更できる**機能を追加しました。
「キーワードで検索」「タグから絞り込む」「カテゴリーから絞り込む」の並び順を、サイトの方針に合わせて自由に設定できます。

- **対象バージョン:** 1.8.0
- **必須環境:** WordPress 6.0 以上 / PHP 7.4 以上
- **推奨環境:** WordPress 6.8 / PHP 8.2

---

## ✨ 新機能

### 表示順設定

管理画面（「Tag Search」→「表示順設定」）に、検索モーダル内ブロックの並び替え機能を追加しました。

- 対象ブロック:
  1. **キーワードで検索**
  2. **タグから絞り込む**（タグカテゴリー）
  3. **カテゴリーから絞り込む**
- 各項目の **▲（上へ）／▼（下へ）** ボタンで並び替え
- 設定した順番が、フロントの検索モーダルにそのまま反映されます
- 未設定の場合は既定順（キーワード → タグ → カテゴリー）

---

## 🔼 アップグレード時の注意

- 既存の設定はそのまま引き継がれます。表示順を変更しない場合は、これまでと同じ並び順で表示されます。

---

## 📦 インストール / アップデート

1. `advanced-tag-search.zip` をダウンロード
2. WordPress 管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ZIP を選択してアップロードし、「今すぐインストール」→「プラグインを有効化」

詳細は `INSTALL.md` / `USAGE_GUIDE.md` を参照してください。
