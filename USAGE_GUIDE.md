# Advanced Tag Search - 使用ガイド

## プラグインの概要

Advanced Tag Searchは、美しく使いやすい検索機能をWordPressサイトに追加するプラグインです。訪問者は、モーダルウィンドウでキーワード・タグ・カテゴリーを組み合わせて、目的のコンテンツを簡単に見つけることができます。

## 主な機能

### 1. 検索窓
- トップページなどに配置できる美しい検索窓
- タイトルとプレースホルダーテキストを管理画面でカスタマイズ可能
- 検索ボタン付き

### 2. モーダルウィンドウ
- 検索窓をクリックすると開く
- 上部のキーワード入力欄、タグ一覧、カテゴリー一覧をまとめて表示
- 複数タグ・複数カテゴリーの選択が可能
- レスポンシブデザイン

### 3. キーワード検索
モーダル上部の入力欄にキーワードを入れると、WordPress標準のサイト内検索（`?s=キーワード`）を実行します。Enterキーでも検索できます。タグ・カテゴリーの選択と組み合わせることもできます。

### 4. タグ絞り込み
タグをカテゴリー別に表示します。標準では以下の3カテゴリーが用意されています。

- ジャンルから絞り込む
- エリアから絞り込む
- キーワードから絞り込む

タグを複数選択すると、いずれかのタグを含む記事（OR検索）がまとめて表示されます。カテゴリーは管理画面で追加・削除できます。

### 5. カテゴリー絞り込み
WordPressのカテゴリーで絞り込めます（AND）。モーダルに表示するカテゴリーは管理画面で選択できます。

### 6. クイックリンク
検索窓の下に、投稿のあるカテゴリーへのアーカイブリンクを表示できます。表示するカテゴリーは管理画面で選択できます。

### 7. 件数表示
選択したタグ・カテゴリー・キーワードに一致する記事数をAjaxで取得し、絞り込みボタンに「選択した条件で絞り込む (N件)」と表示します。選択や入力のたびに更新されます。

### 8. 表示順設定
検索モーダル内のブロック（キーワード／タグ／カテゴリー）の表示順を、管理画面の▲▼ボタンで自由に変更できます。

### 9. 管理画面
WordPress管理画面から簡単に設定を変更できます。

## インストール手順

### ステップ1: プラグインのインストール

1. `advanced-tag-search.zip` をダウンロード
2. WordPress管理画面にログイン
3. 「プラグイン」→「新規追加」をクリック
4. 「プラグインのアップロード」をクリック
5. ZIPファイルを選択してアップロード
6. 「今すぐインストール」をクリック
7. 「プラグインを有効化」をクリック

> **注意**: 有効化にはPHP 7.4以上 / WordPress 6.0以上が必要です。要件を満たさない場合は有効化が中止され、管理画面に通知が表示されます。

### ステップ2: 初期設定

1. WordPress管理画面で「Tag Search」メニューをクリック
2. 基本設定を行う:
   - 検索窓のタイトル
   - 検索窓のプレースホルダー
   - モーダルのタイトル

### ステップ3: 色設定

1. 虫眼鏡アイコンの色をカラーピッカーで設定
2. 検索ボタンの色をカラーピッカーで設定

### ステップ4: タグカテゴリーの設定

1. 各カテゴリーに表示するタグを、WordPressに登録済みのタグからチェックボックスで選択
2. 「全選択」「全解除」ボタンでまとめて選択・解除が可能
3. 「＋ カテゴリーを追加」ボタンでカテゴリーを増やせます
4. 不要なカテゴリーは「このカテゴリーを削除」ボタンで削除できます

> カテゴリー名・タグともに空のカテゴリーは、保存時に自動的に除外されます。

### ステップ5: カテゴリー絞り込みの設定

1. 検索モーダルの「カテゴリーから絞り込む」に表示するカテゴリーをチェックボックスで選択
2. 「全選択」「全解除」が利用可能
3. 未設定の場合は全カテゴリーが表示され、すべて未選択の場合はカテゴリー絞り込みが非表示になります

### ステップ6: クイックリンクの設定

1. 検索窓の下に表示するクイックリンクを、投稿のあるカテゴリーからチェックボックスで選択
2. 「全選択」「全解除」が利用可能
3. 未設定の場合は投稿のある全カテゴリーが表示され、すべて未選択の場合はクイックリンクが非表示になります
4. リンク先は各カテゴリーのアーカイブページです

### ステップ7: 設定を保存

「設定を保存」ボタンをクリックして設定を保存します。

## ショートコードの使い方

### 基本的な使用

ページや投稿に以下のショートコードを追加:

```
[advanced_search]
```

### カスタマイズ例

#### 例1: プレースホルダーを変更

```
[advanced_search placeholder="記事を検索"]
```

#### 例2: クイックリンクを非表示

```
[advanced_search show_quick_links="false"]
```

#### 例3: 両方をカスタマイズ

```
[advanced_search placeholder="記事を検索" show_quick_links="false"]
```

### クイックリンクのみ表示

```
[search_quick_links]
```

## トップページへの配置

### 方法1: ページエディタで配置

1. 「固定ページ」→「トップページ」を編集
2. ショートコードブロックを追加
3. `[advanced_search]` を入力
4. ページを更新

### 方法2: テーマファイルで配置

テーマの `front-page.php` または `home.php` を編集:

```php
<div class="search-section">
    <h2>川越の気になる!を探す</h2>
    <?php echo do_shortcode('[advanced_search]'); ?>
</div>
```

## タグとカテゴリーの準備

### タグの作成

1. 「投稿」→「タグ」をクリック
2. 必要なタグを作成し、記事に割り当てる
3. 管理画面の「タグカテゴリー設定」で、作成済みのタグをチェックボックスで選択

> タグは「WordPressに登録済みのタグから選択」する方式です。タグ名を手入力する必要はありません。

### カテゴリーの作成

1. 「投稿」→「カテゴリー」をクリック
2. 必要なカテゴリーを作成
3. 記事にカテゴリーを割り当て
4. 「カテゴリー絞り込み設定」や「クイックリンク設定」で表示対象を選択

## 検索機能の動作フロー

### ユーザーの操作フロー

1. **検索窓をクリック**
   - モーダルウィンドウが開く

2. **条件を指定**
   - キーワードを入力（サイト内検索）
   - 目的のタグをクリックして複数選択（選択したタグは色が変わる）
   - WordPressカテゴリーで絞り込み
   - キーワード・タグ・カテゴリーは組み合わせて指定可能

3. **件数を確認**
   - ボタンに「選択した条件で絞り込む (N件)」と一致する記事数が表示される（選択・入力のたびに更新）

4. **絞り込みボタンをクリック**
   - 選択した条件に一致する記事一覧ページへ遷移

5. **検索結果を閲覧**
   - 該当する記事が一覧表示される

### 検索URLの形式

検索条件は次のクエリパラメータの組み合わせとしてURLに付与されます。

- `?s=キーワード` - サイト内検索
- `?tag=タグ1,タグ2` - タグ絞り込み（複数指定時はAND）
- `?ats_category=カテゴリースラッグ` - カテゴリー絞り込み

例:

```
https://yoursite.com/?s=川越&tag=sweets,cafe&ats_category=shops
```

複数のタグを選択した場合、カンマ区切りでURLに含まれ、いずれかのタグを含む記事（OR）がまとめて表示されます。タグとカテゴリーを併用した場合は、タグ条件（OR）とカテゴリー条件（OR）の両方を満たす記事（AND）が表示されます。

## カスタマイズ方法

### デザインのカスタマイズ

虫眼鏡アイコンと検索ボタンの色は管理画面のカラーピッカーで変更できます。さらに細かく調整したい場合は、テーマの `style.css` または追加CSSに以下のコードを追加してください。

#### 検索ボタンの色を変更

```css
.ats-search-button {
    background: #your-color !important;
}

.ats-search-button:hover {
    background: #your-hover-color !important;
}
```

#### 選択されたタグの色を変更

```css
.ats-tag.selected {
    background: #your-bg-color !important;
    border-color: #your-border-color !important;
    color: #your-text-color !important;
}
```

#### 絞り込みボタンの色を変更

```css
.ats-search-submit {
    background: #your-bg-color !important;
    color: #your-text-color !important;
}

.ats-search-submit:hover {
    background: #your-hover-color !important;
    color: #your-hover-text-color !important;
}
```

#### 全体の色を変更（CSS変数を使用）

```css
:root {
    --ats-primary-color: #A0616A;      /* メインカラー */
    --ats-secondary-color: #E8B4B8;    /* セカンダリカラー */
    --ats-text-color: #333;            /* テキストカラー */
    --ats-bg-color: #FFF;              /* 背景カラー */
}
```

### 検索窓のサイズを変更

```css
.ats-search-container {
    max-width: 1000px; /* デフォルトは800px */
}
```

### モーダルのサイズを変更

```css
.ats-modal {
    max-width: 900px; /* デフォルトは700px */
}
```

## よくある質問

### Q1: タグが表示されません

**A**: 以下を確認してください:
1. 管理画面の「タグカテゴリー設定」でタグが選択されているか
2. WordPressにタグが登録されているか
3. 選択したタグが付いた公開記事があるか

### Q2: モーダルが開きません

**A**: 以下を確認してください:
1. jQueryが正しく読み込まれているか
2. ブラウザのコンソールでJavaScriptエラーがないか
3. 他のプラグインとの競合がないか

### Q3: 検索結果が表示されません

**A**: 以下を確認してください:
1. 記事にタグやカテゴリーが割り当てられているか
2. 記事が公開状態になっているか
3. キーワード・タグ・カテゴリーを複数指定している場合、すべての条件を満たす記事があるか（AND検索のため）

### Q4: デザインが崩れます

**A**: 以下を試してください:
1. ブラウザのキャッシュをクリア
2. テーマのCSSとの競合を確認
3. プラグインのCSSが正しく読み込まれているか確認

### Q5: 複数のタグで検索できますか?

**A**: はい、モーダルで複数のタグを選択できます。いずれかのタグを含む記事（OR検索）がまとめて表示されます。タグとカテゴリー、キーワードを組み合わせることもできます。

### Q6: カテゴリーを追加できますか?

**A**: はい。管理画面の「タグカテゴリー設定」にある「＋ カテゴリーを追加」ボタンでカテゴリーを追加できます。不要なカテゴリーは「このカテゴリーを削除」ボタンで削除できます。

### Q7: 検索結果ページをカスタマイズできますか?

**A**: 検索結果ページは、WordPressのテーマの `search.php`、`tag.php`、`archive.php` などのテンプレートで制御されます。これらのファイルを編集することでカスタマイズできます。

## トラブルシューティング

### 問題: プラグインが有効化できない

**解決策**:
1. PHPバージョンが7.4以上か確認
2. WordPressバージョンが6.0以上か確認
3. サーバーのエラーログを確認

> 設定画面の「動作環境」セクションで、現在・必須・推奨のバージョンと判定（✓ / △ / ✕）を確認できます。

### 問題: ショートコードが表示されない

**解決策**:
1. プラグインが有効化されているか確認
2. ショートコードのスペルが正しいか確認
3. テーマがショートコードをサポートしているか確認

### 問題: スタイルが適用されない

**解決策**:
1. ブラウザのキャッシュをクリア
2. WordPressのキャッシュプラグインをクリア
3. CSSファイルが正しく読み込まれているか確認（開発者ツールで確認）

## ベストプラクティス

### 1. タグの命名規則

- 短く分かりやすい名前を使用
- 一貫性のある命名規則を採用
- 重複を避ける

### 2. タグカテゴリーの整理

- 各カテゴリーに適切な数のタグを配置（5〜15個程度）
- 使用頻度の高いタグを優先的に配置
- 定期的に見直して最適化

### 3. クイックリンクの活用

- 人気のカテゴリーへのリンクを配置
- 数を絞る（多すぎると選択が困難）
- 投稿のあるカテゴリーを選ぶ

### 4. パフォーマンス最適化

- 不要なタグを削除
- 画像の最適化
- キャッシュプラグインの使用

## サポート情報

### 必要な情報

問題が発生した場合、以下の情報を準備してください:

- WordPress バージョン
- PHP バージョン
- 使用しているテーマ
- 有効化している他のプラグイン
- エラーメッセージ（あれば）
- ブラウザとそのバージョン

### デバッグモード

問題を特定するために、WordPressのデバッグモードを有効にできます:

`wp-config.php` に以下を追加:

```php
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
define('WP_DEBUG_DISPLAY', false);
```

エラーログは `wp-content/debug.log` に記録されます。

## アップデート情報

プラグインのアップデートは、WordPress管理画面の「プラグイン」メニューから確認できます。

### アップデート前の注意事項

1. サイトのバックアップを取る
2. テスト環境で動作確認を行う
3. カスタマイズしたCSSをバックアップ

## まとめ

Advanced Tag Searchプラグインを使用することで、訪問者はキーワード・タグ・カテゴリーを組み合わせて直感的にコンテンツを見つけることができます。適切な設定とカスタマイズにより、サイトのユーザビリティを大幅に向上させることができます。
