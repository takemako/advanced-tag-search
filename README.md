# Advanced Tag Search - WordPressプラグイン

## 概要

**Advanced Tag Search**は、美しく使いやすい検索機能をWordPressサイトに追加するプラグインです。訪問者は、モーダルウィンドウでキーワード・タグ・カテゴリーを組み合わせて、目的のコンテンツを簡単に見つけることができます。

- バージョン: 1.7.0
- 作者: Makoto Takei
- ライセンス: GPLv2 or later

## 特徴

### デザイン
- モダンで洗練されたUI
- レスポンシブデザイン対応
- カスタマイズ可能なカラー（虫眼鏡アイコン・検索ボタンの色）
- スムーズなアニメーション効果

### 機能
- **検索窓**: トップページなどに配置できる美しい検索窓（タイトル・プレースホルダーを管理画面で設定可能）
- **モーダルウィンドウ**: クリックで開くモーダルで、キーワード・タグ・カテゴリーをまとめて指定
- **キーワード検索**: モーダル上部の入力欄からWordPress標準のサイト内検索（`?s=キーワード`）を実行
- **タグ絞り込み**: タグをカテゴリー別に表示し、複数選択でOR検索（いずれかのタグを含む記事をまとめて表示）
- **カテゴリー絞り込み**: WordPressのカテゴリーでAND絞り込み
- **クイックリンク**: 投稿のあるカテゴリーへのアーカイブリンクを検索窓の下に表示
- **件数表示**: 選択した条件に一致する記事数をAjaxでリアルタイム表示
- **表示順設定**: モーダル内ブロック（キーワード／タグ／カテゴリー）の表示順を管理画面で変更可能
- **管理画面**: WordPress管理画面から簡単に設定変更
- **ショートコード**: どこにでも簡単に配置可能

### タグカテゴリー
タグはカテゴリー別に整理して表示できます。標準では以下の3カテゴリーが用意されています。

1. **ジャンルから絞り込む**
2. **エリアから絞り込む**
3. **キーワードから絞り込む**

各カテゴリーに表示するタグは、WordPressに登録済みのタグからチェックボックスで選択します。管理画面の「＋ カテゴリーを追加」ボタンでカテゴリーを増やしたり、「このカテゴリーを削除」で減らしたりできます。

詳細なシステム要件は `REQUIREMENTS.md` をご覧ください。

## インストール

### 方法1: ZIPファイルからインストール（推奨）

1. `advanced-tag-search.zip` をダウンロード
2. WordPress管理画面にログイン
3. 「プラグイン」→「新規追加」をクリック
4. 「プラグインのアップロード」をクリック
5. ZIPファイルを選択してアップロード
6. 「今すぐインストール」をクリック
7. 「プラグインを有効化」をクリック

### 方法2: FTPでアップロード

1. `advanced-tag-search` フォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面で「Advanced Tag Search」を有効化

詳細なインストール手順は `INSTALL.md` を参照してください。

## 使い方

### 基本的な使用

ページや投稿に以下のショートコードを追加:

```
[advanced_search]
```

### カスタマイズ

```
[advanced_search placeholder="検索してみる" show_quick_links="false"]
```

### テーマファイルで使用

```php
<?php echo do_shortcode('[advanced_search]'); ?>
```

詳細な使用方法は `USAGE_GUIDE.md` を参照してください。

## ショートコード

### `[advanced_search]`
検索窓、クイックリンク、モーダルを表示

**パラメータ:**
- `placeholder` - 検索窓のプレースホルダー（デフォルト: "タグから探してみる"）
- `show_quick_links` - クイックリンクの表示（デフォルト: true）

**例:**
```
[advanced_search]
[advanced_search placeholder="記事を検索"]
[advanced_search show_quick_links="false"]
```

### `[search_quick_links]`
クイックリンクのみを表示（引数なし）

**例:**
```
[search_quick_links]
```

## 設定

WordPress管理画面の「Tag Search」メニューから以下の設定が可能:

1. **基本設定**
   - 検索窓のタイトル
   - 検索窓のプレースホルダー
   - モーダルのタイトル

2. **色設定**
   - 虫眼鏡アイコンの色（カラーピッカー）
   - 検索ボタンの色（カラーピッカー）

3. **表示順設定**
   - 検索モーダル内のブロック（キーワード／タグ／カテゴリー）の表示順を▲▼ボタンで変更

4. **タグカテゴリー設定**
   - 各カテゴリーに表示するタグを、WordPressに登録済みのタグからチェックボックスで選択
   - 各リストに「全選択」「全解除」ボタン
   - 「＋ カテゴリーを追加」「このカテゴリーを削除」でカテゴリーを増減（カテゴリー名・タグともに空の項目は保存時に除外）

5. **カテゴリー絞り込み設定**
   - 検索モーダルに表示するWordPressカテゴリーをチェックボックスで選択（全選択/全解除対応）
   - 未設定の場合は全カテゴリーを表示、すべて未選択の場合はカテゴリー絞り込みを非表示

6. **クイックリンク設定**
   - 検索窓の下に表示するクイックリンクを、投稿のあるカテゴリーからチェックボックスで選択（全選択/全解除対応）
   - 未設定の場合は投稿のある全カテゴリーを表示、すべて未選択の場合はクイックリンクを非表示
   - リンク先は各カテゴリーのアーカイブページ

7. **動作環境**
   - 現在・必須・推奨のPHP/WordPressバージョンと判定を表示

## カスタマイズ

### CSSのカスタマイズ

虫眼鏡アイコンと検索ボタンの色は管理画面のカラーピッカーで変更できます。さらに細かく調整したい場合は、テーマの `style.css` や追加CSSに以下のようなコードを追加してください。

```css
/* 検索ボタンの色を変更 */
.ats-search-button {
    background: #your-color !important;
}

/* 選択されたタグの色を変更 */
.ats-tag.selected {
    background: #your-bg-color !important;
    border-color: #your-border-color !important;
}

/* CSS変数で全体の色を変更 */
:root {
    --ats-primary-color: #A0616A;
    --ats-secondary-color: #E8B4B8;
}
```

## 動作フロー

1. **ユーザーが検索窓をクリック**
   - モーダルウィンドウが開く

2. **条件を指定**
   - キーワードを入力（サイト内検索）
   - カテゴリーから目的のタグをクリックして複数選択（AND）
   - WordPressカテゴリーで絞り込み（AND）
   - キーワード・タグ・カテゴリーは組み合わせて指定可能

3. **件数を確認**
   - 選択した条件に一致する記事数がボタンに「選択した条件で絞り込む (N件)」と表示される（選択・入力のたびに更新）

4. **絞り込みボタンをクリック**
   - 選択した条件に一致する記事一覧ページへ遷移

5. **検索結果を閲覧**
   - 該当する記事が表示される

## トラブルシューティング

### モーダルが開かない
- jQueryが正しく読み込まれているか確認
- ブラウザのコンソールでエラーを確認
- 他のプラグインとの競合を確認

### タグが表示されない
- 管理画面でタグカテゴリーにタグが選択されているか確認
- WordPressにタグが登録されているか確認
- 選択したタグが付いた公開記事があるか確認

### デザインが崩れる
- ブラウザのキャッシュをクリア
- テーマのCSSとの競合を確認
- プラグインのCSSが正しく読み込まれているか確認

詳細は `USAGE_GUIDE.md` のトラブルシューティングセクションを参照してください。

## 検索URLの形式

検索条件は次のクエリパラメータの組み合わせとしてURLに付与されます。

- `?s=キーワード` - サイト内検索（WordPress標準）
- `?tag=タグ1,タグ2` - タグ絞り込み（複数指定時はAND）
- `?ats_category=カテゴリースラッグ` - カテゴリー絞り込み

例:

```
?s=川越&tag=sweets,cafe&ats_category=shops
```

## 開発者向け情報

### フック

プラグインは以下のWordPressフックを使用:

- `wp_enqueue_scripts` - フロントエンドスクリプトの読み込み
- `admin_enqueue_scripts` - 管理画面スクリプトの読み込み
- `init` - ショートコードの登録
- `admin_menu` - 管理メニューの追加
- `pre_get_posts` - タグ・カテゴリーのAND絞り込みクエリ制御
- `wp_ajax_*` / `wp_ajax_nopriv_*` - Ajax処理（タグ取得・記事数取得）
