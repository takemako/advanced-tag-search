# Advanced Tag Search - WordPressプラグイン

## 概要

**Advanced Tag Search**は、美しく使いやすい検索機能をWordPressサイトに追加するプラグインです。訪問者は、モーダルウィンドウでタグを選択して、目的のコンテンツを簡単に見つけることができます。

## 特徴

### デザイン
- モダンで洗練されたUI
- レスポンシブデザイン対応
- カスタマイズ可能なカラースキーム
- スムーズなアニメーション効果

### 機能
- **検索窓**: トップページに配置できる美しい検索窓
- **モーダルウィンドウ**: タグをカテゴリー別に整理して表示
- **複数タグ選択**: 複数のタグで絞り込み検索が可能
- **クイックリンク**: よく使うカテゴリーへの直接リンク
- **管理画面**: WordPress管理画面から簡単に設定変更
- **ショートコード**: どこにでも簡単に配置可能

### タグカテゴリー
以下の6つのカテゴリーでタグを分類:
1. **ジャンルから絞り込む** - 食べ歩き、レストラン、カフェなど
2. **エリアから絞り込む** - 地域や駅周辺など
3. **キーワードから絞り込む** - 料理名や食材など
4. **時間帯から絞り込む** - モーニング、ランチ、ディナー
5. **季節から絞り込む** - 春、夏、秋、冬
6. **その他から絞り込む** - カスタムタグ

## インストール

### 方法1: ZIPファイルからインストール（推奨）

1. `advanced-tag-search.zip` をダウンロード
2. WordPress管理画面にログイン
3. 「プラグイン」→「新規追加」をクリック
4. 「プラグインのアップロード」をクリック
5. ZIPファイルを選択してアップロード
6. 「今すぐインストール」をクリック
7. 「プラグインを有効化」をクリック

### 方法2: FTPでアップロード

1. `advanced-tag-search` フォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面で「Advanced Tag Search」を有効化

詳細なインストール手順は `INSTALL.md` を参照してください。

## 使い方

### 基本的な使用

ページや投稿に以下のショートコードを追加:

```
[advanced_search]
```

### カスタマイズ

```
[advanced_search placeholder="検索してみる" show_quick_links="false"]
```

### テーマファイルで使用

```php
<?php echo do_shortcode('[advanced_search]'); ?>
```

詳細な使用方法は `USAGE_GUIDE.md` を参照してください。

## ショートコード

### `[advanced_search]`
検索窓、クイックリンク、モーダルを表示

**パラメータ:**
- `placeholder` - 検索窓のプレースホルダー（デフォルト: "タグから探してみる"）
- `show_quick_links` - クイックリンクの表示（デフォルト: true）

**例:**
```
[advanced_search]
[advanced_search placeholder="記事を検索"]
[advanced_search show_quick_links="false"]
```

### `[search_quick_links]`
クイックリンクのみを表示

**例:**
```
[search_quick_links]
```

## 設定

### 管理画面

WordPress管理画面の「Tag Search」メニューから以下の設定が可能:

1. **基本設定**
   - 検索窓のプレースホルダー
   - モーダルのタイトル

2. **タグカテゴリー設定**
   - 各カテゴリーのタグをカンマ区切りで設定

3. **クイックリンク設定**
   - リンクテキストとURLを設定

## カスタマイズ

### CSSのカスタマイズ

テーマの `style.css` に以下のコードを追加:

```css
/* 検索ボタンの色を変更 */
.ats-search-button {
    background: #your-color !important;
}

/* 選択されたタグの色を変更 */
.ats-tag.selected {
    background: #your-bg-color !important;
    border-color: #your-border-color !important;
}

/* CSS変数で全体の色を変更 */
:root {
    --ats-primary-color: #A0616A;
    --ats-secondary-color: #E8B4B8;
}
```

## 動作フロー

1. **ユーザーが検索窓をクリック**
   - モーダルウィンドウが開く

2. **タグを選択**
   - カテゴリーから目的のタグをクリック
   - 複数のタグを選択可能

3. **絞り込みボタンをクリック**
   - 選択したタグを含む記事一覧ページへ遷移

4. **検索結果を閲覧**
   - 該当する記事が表示される

## トラブルシューティング

### モーダルが開かない
- jQueryが正しく読み込まれているか確認
- ブラウザのコンソールでエラーを確認
- 他のプラグインとの競合を確認

### タグが表示されない
- 管理画面でタグカテゴリーが設定されているか確認
- WordPressにタグが登録されているか確認
- タグ名が完全に一致しているか確認

### デザインが崩れる
- ブラウザのキャッシュをクリア
- テーマのCSSとの競合を確認
- プラグインのCSSが正しく読み込まれているか確認

詳細は `USAGE_GUIDE.md` のトラブルシューティングセクションを参照してください。
