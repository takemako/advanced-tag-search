# Advanced Tag Search - クイックスタートガイド

## 5分で始める!

このガイドでは、プラグインを最速でインストールして動作させる方法を説明します。

## ステップ1: インストール（2分）

1. `advanced-tag-search.zip` をダウンロード
2. WordPress管理画面 → 「プラグイン」→「新規追加」
3. 「プラグインのアップロード」をクリック
4. ZIPファイルを選択してアップロード
5. 「今すぐインストール」→「プラグインを有効化」

✅ インストール完了!

## ステップ2: 基本設定（1分）

1. WordPress管理画面 → 「Tag Search」メニュー
2. 設定はデフォルトのままでOK（後で変更可能）
3. 「設定を保存」をクリック

✅ 設定完了!

## ステップ3: ページに配置（1分）

### トップページに配置する場合

1. 「固定ページ」→ トップページを編集
2. ショートコードブロックを追加
3. 以下を入力:
   ```
   [advanced_search]
   ```
4. 「更新」をクリック

✅ 配置完了!

## ステップ4: タグを作成（1分）

1. 「投稿」→「タグ」
2. 以下のタグを作成（例）:
   - 食べ歩き
   - レストラン
   - カフェ
   - スイーツ
3. 記事にタグを割り当て

✅ タグ作成完了!

## 完了!

サイトのトップページにアクセスして、検索窓が表示されることを確認してください。

検索窓をクリックすると、モーダルウィンドウが開き、タグを選択できます。

## 次のステップ

### カスタマイズしたい場合

詳細なカスタマイズ方法は `USAGE_GUIDE.md` を参照してください。

### タグカテゴリーを編集したい場合

1. 「Tag Search」メニュー
2. 各カテゴリーのタグをカンマ区切りで編集
3. 「設定を保存」

### デザインを変更したい場合

テーマの追加CSSに以下を追加:

```css
/* 検索ボタンの色を変更 */
.ats-search-button {
    background: #your-color !important;
}
```

## トラブルシューティング

### 検索窓が表示されない
→ ショートコードが正しく入力されているか確認

### モーダルが開かない
→ ブラウザのキャッシュをクリア

### タグが表示されない
→ 管理画面でタグカテゴリーが設定されているか確認
