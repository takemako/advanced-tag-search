# Advanced Tag Search v1.8.3 - リリースノート

## 📋 概要

v1.8.3 では、**記事数が0件のタグ・カテゴリーを選択不可にする**改善を行いました。
記事のないタグを選んでしまい、検索しても0件になってしまう問題を視覚的に防ぎます。

- **対象バージョン:** 1.8.3
- **必須環境:** WordPress 6.0 以上 / PHP 7.4 以上
- **推奨環境:** WordPress 6.8 / PHP 8.2

---

## ✨ 変更内容

### 記事のないタグ・カテゴリーを選択不可に

検索モーダルの各ボタンを、記事数に応じて表示を切り替えるようにしました。

- **記事のあるタグ・カテゴリー:** 通常表示。クリックで選択／解除。
- **記事のないタグ・カテゴリー:** **薄く（opacity 0.4）表示**し、クリックしても選択されません。
  - マウスオーバー時のカーソルは `not-allowed`
  - `disabled` 属性および `aria-disabled="true"` を付与（スクリーンリーダー対応）
  - ツールチップに「このタグに記事はありません」と表示

これにより、Kawagoe.fun のような UI で、**選べるタグが一目で分かる**ようになります。

### 対象範囲

- タグカテゴリー内のタグ（管理画面で設定したタグ）
- 「カテゴリーから絞り込む」のWordPressカテゴリーボタン

### 件数の判定

- WordPress側の投稿数（`count`）が **0** または、そもそも該当タグがWordPressに登録されていない場合は「記事なし」と判定します。
- 記事を新規追加・公開してタグが付くと、自動的に選択可能になります。

---

## 🔼 アップグレード時の注意

- 設定の変更は不要です。プラグインを更新するだけで反映されます。
- 表示が以前と変わらない場合は、ブラウザのキャッシュをクリアしてください（CSSが古い可能性があります）。

---

## 📦 インストール / アップデート

1. `advanced-tag-search.zip` をダウンロード
2. WordPress 管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ZIP を選択してアップロードし、「今すぐインストール」→「プラグインを有効化」
