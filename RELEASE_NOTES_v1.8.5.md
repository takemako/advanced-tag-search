# Advanced Tag Search v1.8.5 - リリースノート

## 📋 概要

v1.8.5 では、検索モーダルでの**タグ選択を1つだけに制限**しました（単一選択）。シンプルで分かりやすい絞り込みを優先しています。

- **対象バージョン:** 1.8.5
- **必須環境:** WordPress 6.0 以上 / PHP 7.4 以上
- **推奨環境:** WordPress 6.8 / PHP 8.2

---

## 🔄 変更内容

### タグ選択を単一選択に

- 検索モーダルで**選択できるタグは1つのみ**になりました。
- 別のタグをクリックすると、**以前の選択は自動的に解除**されます。
- 選択中のタグをもう一度クリックすると、解除されます。

これにより、絞り込み結果が分かりやすくなり、複数選択時に発生していた挙動の混乱もなくなります。

### 維持されるもの

- キーワード検索とタグ選択の組み合わせは引き続き可能です。
- タグ選択 → 該当タグのアーカイブページへ遷移、という基本フローは変わりません。
- 記事のないタグは薄く表示・選択不可、絞り込みボタンに件数表示などはそのまま動作します。

---

## 🔼 アップグレード時の注意

- 設定の変更は不要です。プラグインを更新するだけで反映されます。
- 表示が以前と変わらない場合は、ブラウザのキャッシュをクリアしてください。

---

## 📦 インストール / アップデート

1. `advanced-tag-search.zip` をダウンロード
2. WordPress 管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ZIP を選択してアップロードし、「今すぐインストール」→「プラグインを有効化」
