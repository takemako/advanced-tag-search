# Advanced Tag Search - システム要件

## 推奨環境

プラグインを快適にご利用いただくための推奨環境です。

### WordPress

- **最小バージョン**: WordPress 6.0
- **推奨バージョン**: WordPress 6.8

### PHP

- **最小バージョン**: PHP 7.4
- **推奨バージョン**: PHP 8.2以上

**PHPバージョンの確認方法:**
1. WordPress管理画面 → 「ツール」→「サイトヘルス」
2. 「情報」タブ → 「サーバー」セクションで確認

### データベース

- **MySQL**: 5.7以上
- **MariaDB**: 10.3以上

### Webサーバー

- Apache 2.4以上（mod_rewrite有効）
- Nginx 1.18以上

## 最小要件

プラグインが動作する最小限の要件です。

| 項目 | 最小バージョン |

## PHPの必須拡張機能

以下のPHP拡張機能が必要です（通常は標準でインストールされています）:

- `json` - JSON処理
- `mbstring` - マルチバイト文字列処理
- `mysqli` または `pdo_mysql` - データベース接続

## WordPressの必須機能

- REST API有効
- パーマリンク設定が「基本」以外

## パフォーマンス推奨

### サーバー設定

- **メモリ制限**: 128MB以上（256MB推奨）
- **最大実行時間**: 30秒以上
- **最大アップロードサイズ**: 8MB以上

#### 対応プラグイン

- WooCommerce

## 非推奨環境

以下の環境では動作しない、または問題が発生する可能性があります:

- **PHP 7.3以下**: セキュリティサポート終了
- **WordPress 4.x**: 古すぎるバージョン
- **MySQL 5.5以下**: パフォーマンス問題
- **Internet Explorer**: サポート対象外

## バージョンアップグレード推奨

### PHPのアップグレード

PHP 7.4は2022年11月にセキュリティサポートが終了しています。
セキュリティとパフォーマンスのため、PHP 8.2以上へのアップグレードを強く推奨します。

**アップグレードのメリット:**
- セキュリティの向上
- パフォーマンスの改善（最大30%高速化）
- 最新機能の利用

**アップグレード方法:**
レンタルサーバーの管理画面からPHPバージョンを変更できます。
詳細はご利用のホスティング会社にお問い合わせください。

### WordPressのアップグレード

WordPress 5.xは古いバージョンです。
セキュリティとパフォーマンスのため、WordPress 6.x以上へのアップグレードを推奨します。

**アップグレード方法:**
1. WordPress管理画面 → 「ダッシュボード」→「更新」
2. 「今すぐ更新」をクリック

## トラブルシューティング

### バージョンが古い場合

**エラーメッセージ:**
```
このプラグインにはPHP 7.4以上が必要です
```

**解決方法:**
1. ホスティング会社の管理画面でPHPバージョンを確認
2. PHP 7.4以上にアップグレード
3. プラグインを再度有効化

### 互換性の確認

**WordPress管理画面で確認:**
1. 「ツール」→「サイトヘルス」
2. 「情報」タブで現在の環境を確認
3. 推奨環境と比較
