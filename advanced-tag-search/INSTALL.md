# Advanced Tag Search - インストール手順

## 概要

このプラグインは、高度な検索機能をWordPressサイトに追加します。

## システム要件

- WordPress 6.0以上（6.8以上を推奨）
- PHP 7.4以上（8.2以上を推奨）
- jQuery（WordPressに標準搭載）

## インストール方法

### 方法1: ZIPファイルからインストール（推奨）

1. `advanced-tag-search.zip` ファイルをダウンロード
2. WordPress管理画面にログイン
3. 「プラグイン」→「新規追加」をクリック
4. 「プラグインのアップロード」をクリック
5. ZIPファイルを選択してアップロード
6. 「今すぐインストール」をクリック
7. インストール完了後、「プラグインを有効化」をクリック

### 方法2: FTPでアップロード

1. `advanced-tag-search` フォルダをFTPクライアントで `/wp-content/plugins/` ディレクトリにアップロード
2. WordPress管理画面にログイン
3. 「プラグイン」メニューから「Advanced Tag Search」を有効化

## 初期設定

### 1. 基本設定

1. WordPress管理画面で「Tag Search」メニューをクリック
2. 以下の項目を設定:
   - **検索窓のプレースホルダー**: 検索窓に表示されるテキスト（デフォルト: "タグから探してみる"）
   - **モーダルのタイトル**: モーダルウィンドウのタイトル（デフォルト: "タグ検索"）

### 2. タグカテゴリー設定

タグを以下のカテゴリーに分類できます:

- **ジャンルから絞り込む**: 食べ歩き、レストラン、カフェなど
- **エリアから絞り込む**: 地域や駅周辺など
- **キーワードから絞り込む**: 料理名や食材など
- **時間帯から絞り込む**: モーニング、ランチ、ディナー
- **季節から絞り込む**: 春、夏、秋、冬
- **その他から絞り込む**: カスタムタグ

各カテゴリーには、WordPressに登録済みのタグがチェックボックスで一覧表示されます。そのカテゴリーに表示したいタグにチェックを入れて保存してください。各リストの上部にある「全選択」「全解除」ボタンで一括操作もできます。

> ※ チェックボックスに表示されるのは「投稿」→「タグ」で登録済みのタグのみです。表示したいタグが無い場合は、先にWordPressでタグを作成してください。

### 3. カテゴリー絞り込み設定

検索モーダルの「カテゴリーから絞り込む」に表示するカテゴリーを、登録済みカテゴリーのチェックボックスから選択できます（「全選択」「全解除」ボタン対応）。

- すべて未選択の場合、モーダルにカテゴリー絞り込みは表示されません。
- 初期状態（未設定）では、すべてのカテゴリーが表示されます。

### 4. クイックリンク設定

クイックリンクは、ブログのカテゴリー（投稿のあるもの）から**自動生成**されます。手動設定は不要です。

- 「投稿」→「カテゴリー」でカテゴリーを編集すると、検索窓の下のクイックリンクにも自動で反映されます。
- リンク先は各カテゴリーのアーカイブページです。

### 5. 設定を保存

すべての設定が完了したら、「設定を保存」ボタンをクリックしてください。

## ショートコードの使用方法

### 基本的な使用

ページや投稿に以下のショートコードを追加:

```
[advanced_search]
```

これで検索窓、クイックリンク、モーダルが表示されます。

### カスタマイズオプション

#### プレースホルダーをカスタマイズ

```
[advanced_search placeholder="検索してみる"]
```

#### クイックリンクを非表示

```
[advanced_search show_quick_links="false"]
```

#### 複数オプションの組み合わせ

```
[advanced_search placeholder="検索してみる" show_quick_links="false"]
```

### クイックリンクのみ表示

```
[search_quick_links]
```

### テーマファイルで使用

PHPファイルで直接使用する場合:

```php
<?php echo do_shortcode('[advanced_search]'); ?>
```

## トップページへの配置方法

### 方法1: ブロックエディタ（Gutenberg）

1. トップページを編集
2. 「ショートコード」ブロックを追加
3. `[advanced_search]` を入力
4. ページを更新

### 方法2: クラシックエディタ

1. トップページを編集
2. エディタに `[advanced_search]` を入力
3. ページを更新

### 方法3: テーマのテンプレートファイル

1. テーマの `front-page.php` または `home.php` を編集
2. 表示したい位置に以下のコードを追加:

```php
<?php echo do_shortcode('[advanced_search]'); ?>
```

3. ファイルを保存

## タグとカテゴリーの準備

プラグインが正しく動作するには、WordPressにタグとカテゴリーが登録されている必要があります。

### タグの作成

1. 「投稿」→「タグ」をクリック
2. プラグインで設定したタグ名と同じ名前でタグを作成
3. 記事にタグを割り当て

### カテゴリーの作成

1. 「投稿」→「カテゴリー」をクリック
2. 必要なカテゴリーを作成
3. 記事にカテゴリーを割り当て

## 検索機能の動作

### タグ検索

1. ユーザーが検索窓をクリック
2. モーダルウィンドウが開く
3. タグを選択
4. 「選択したタグで絞り込む」ボタンをクリック
5. 選択したタグを含む記事一覧ページへ遷移

### 検索URLの形式

```
https://yoursite.com/?tag=タグ1,タグ2
```

複数のタグを選択した場合、カンマ区切りでURLに含まれます。

## カスタマイズ

### CSSのカスタマイズ

テーマの `style.css` に以下のようなコードを追加してデザインをカスタマイズできます:

```css
/* 検索ボタンの色を変更 */
.ats-search-button {
    background: #your-color !important;
}

/* タグの色を変更 */
.ats-tag.selected {
    background: #your-color !important;
    border-color: #your-color !important;
}
```

### 色の変更

プラグインのCSS変数を上書きすることで、全体の色を変更できます:

```css
:root {
    --ats-primary-color: #your-primary-color;
    --ats-secondary-color: #your-secondary-color;
}
```

## トラブルシューティング

### モーダルが開かない

1. ブラウザのコンソールでエラーを確認
2. jQueryが正しく読み込まれているか確認
3. 他のプラグインとの競合がないか確認

### タグが表示されない

1. 管理画面でタグカテゴリーが正しく設定されているか確認
2. WordPressにタグが登録されているか確認

### 検索結果が表示されない

1. 記事にタグが割り当てられているか確認
2. タグ名が完全に一致しているか確認（大文字小文字、スペースに注意）

### デザインが崩れる

1. テーマのCSSとの競合を確認
2. ブラウザのキャッシュをクリア
3. プラグインのCSSが正しく読み込まれているか確認

## サポート

問題が解決しない場合は、以下の情報を含めてサポートにお問い合わせください:

- WordPress バージョン
- PHP バージョン
- 使用しているテーマ
- 有効化している他のプラグイン
- エラーメッセージ（あれば）

## アンインストール

1. 「プラグイン」メニューから「Advanced Tag Search」を無効化
2. 「削除」をクリック
3. プラグインのファイルとデータベースの設定が削除されます
