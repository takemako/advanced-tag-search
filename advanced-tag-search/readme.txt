高度な検索機能を提供するプラグイン。タグやカテゴリーでの絞り込み検索が可能です。

== Description ==

Advanced Tag Searchは、美しく使いやすい検索機能をWordPressサイトに追加するプラグインです。

**動作環境:**

* WordPress: 6.0以上（6.8以上を推奨）
* PHP: 7.4以上（8.2以上を推奨）
* MySQL: 5.7以上 / MariaDB: 10.3以上

**主な機能:**

* モーダルウィンドウでのタグ選択（複数タグのOR検索＝いずれかを含む記事をまとめて表示）
* キーワードによるサイト内検索（タグと組み合わせ可能）
* タグは登録済みタグからチェックボックスで選択（全選択／全解除）
* 記事数が0件のタグは選択不可（薄く表示）
* タグカテゴリーは標準3種類＋「カテゴリーを追加」で増減できる動的UI
* 一致する記事数を絞り込みボタンに表示
* クイックリンク（表示するカテゴリーを選択可能）
* 検索モーダル内ブロック（キーワード／タグ）の表示順を変更可能
* 色のカスタマイズ（カラーピッカー）
* 動作要件（PHP / WordPress）の自動チェック
* レスポンシブデザイン
* ショートコードで簡単設置

**使用方法:**

1. プラグインをインストールして有効化
2. ページやウィジェットに `[advanced_search]` ショートコードを追加
3. 管理画面で設定をカスタマイズ

**ショートコード:**

* `[advanced_search]` - 検索窓とモーダルを表示
* `[advanced_search placeholder="検索してみる" show_quick_links="false"]` - カスタマイズ版
* `[search_quick_links]` - クイックリンクのみ表示

== Installation ==

1. プラグインファイルを `/wp-content/plugins/advanced-tag-search` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューからプラグインを有効化
3. 「Tag Search」メニューから設定を行う
4. ページやウィジェットにショートコードを追加

== Frequently Asked Questions ==

= ショートコードはどこに配置すればいいですか? =

ページ、投稿、ウィジェットなど、ショートコードが使える場所ならどこでも使用できます。
トップページに配置する場合は、テーマのテンプレートファイルで `<?php echo do_shortcode('[advanced_search]'); ?>` を使用してください。

= タグカテゴリーをカスタマイズできますか? =

はい、管理画面の「Tag Search」メニューから、各カテゴリーのタグを自由に編集できます。

= 複数のタグで検索できますか? =

はい、モーダルで複数のタグを選択できます。いずれかのタグを含む記事（OR検索）がまとめて表示されます。

= デザインをカスタマイズできますか? =

はい、CSSでカスタマイズ可能です。プラグインの `assets/css/style.css` を参考に、テーマのCSSでオーバーライドしてください。
